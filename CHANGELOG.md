# Changelog

## v0.1.0 — 2026-04-02

Initial release.

### Features
- CSV upload with auto column detection (date, description, amount)
- Manual column mapping override
- Credit/refund exclusion with smart debit convention detection
- Date range filter
- AI categorization via gpt-5.4-nano (batched by unique merchant)
- Categories: Food & Dining, Groceries, Transport, Shopping, Entertainment & Streaming, Health & Fitness, Travel, Utilities & Bills, Personal Care, Other
- Donut chart by category
- Horizontal bar chart breakdown
- Weekly spending over time bar chart
- Top merchants table
- Category totals table with transaction counts and % of spend
- AI plain-English summary
- Full transaction table with category assignments
- GitHub Actions keepalive for Streamlit Community Cloud
